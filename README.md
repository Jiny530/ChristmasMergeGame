# ChristmasMergeGame (Phaser 2D 최소 초기 세팅)

이 프로젝트는 **Phaser 3(2D 엔진)** 를 CDN으로 불러와서 사용하는,
가장 단순한 형태의 모바일 웹게임 초기 세팅입니다.

번들러(웹팩, Vite 등) 없이도 바로 실행할 수 있도록 구성되어 있습니다.

## 구조

- `index.html` : Phaser를 포함한 HTML 진입 파일
- `main.js` : 기본 게임 설정 및 메인 씬 코드

## 실행 방법

가장 간단한 방법은, 아무 정적 서버로 `index.html` 을 열어보는 것입니다.

예시:

1. VSCode의 Live Server 플러그인 사용
2. 또는 Node.js가 있다면, 아무 간단한 서버 패키지를 사용

```bash
npx serve .
```

이후 브라우저에서 안내된 주소(예: `http://localhost:3000`)로 접속하면 됩니다.
